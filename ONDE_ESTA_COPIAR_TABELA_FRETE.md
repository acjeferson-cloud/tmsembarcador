# 🗺️ ONDE ESTÁ O BOTÃO COPIAR TABELA DE FRETE

## 📍 Localização Exata

O botão de **Copiar Tabela** está dentro da tela de **Tabelas de Frete** de cada transportador.

## 🎯 Caminho Passo a Passo

```
Menu Lateral
  └── 🚚 Transportadores
      └── [Clique em qualquer transportador]
          └── Abas superiores: Visão 360 | Tabelas de Frete
              └── Clique em "Tabelas de Frete"
                  └── Nos cards das tabelas, encontre os ícones:
                      👁️ Ver | 📋 Copiar | ✏️ Editar | 🗑️ Excluir
```

## 📸 Exemplo Visual (baseado no seu print)

No card da tabela, você verá 4 ícones no canto superior direito:

```
┌─────────────────────────────────────────────────┐
│  Tabela Padrão 2025              👁️ 📋 ✏️ 🗑️    │
│  Nome do Transportador                          │
│                                                  │
│  📅 Vigência: 31/12/2024 a 30/12/2025          │
│  💰 1 tarifas cadastradas                       │
└─────────────────────────────────────────────────┘
```

O ícone **📋 (azul)** é o botão de **Copiar Tabela**.

## ✅ Passo a Passo Detalhado

1. **Abra o menu lateral** → Clique em **🚚 Transportadores**

2. **Selecione um transportador** da lista

3. **Na tela do transportador**, você verá abas:
   - Visão 360
   - **Tabelas de Frete** ← CLIQUE AQUI

4. **Você verá cards** com as tabelas de frete deste transportador

5. **No canto superior direito de cada card**, há 4 ícones:
   - 👁️ Ver (cinza)
   - **📋 Copiar (AZUL)** ← ESTE É O BOTÃO!
   - ✏️ Editar (cinza)
   - 🗑️ Excluir (vermelho)

6. **Clique no ícone azul de copiar** 📋

7. **Abrirá um modal** com o formulário de cópia

## 🎨 Características do Botão

- **Cor:** Azul (diferente dos outros que são cinza/vermelho)
- **Posição:** 2º ícone da esquerda para direita
- **Tooltip:** Ao passar o mouse, mostra "Copiar Tabela"
- **Ícone:** Duas folhas sobrepostas (símbolo universal de copiar)

## 💡 O Modal de Cópia

Após clicar no botão azul, abrirá um modal com:

- **Título:** "Copiar Tabela de Frete"
- **Campos:**
  1. Tabela de Origem (dropdown)
  2. Transportador de Destino (dropdown)
  3. Nome da Nova Tabela (texto)
  4. Data Início (data)
  5. Data Fim (data)
- **Botões:** Cancelar | Copiar Tabela

## ⚠️ Importante

- O botão só aparece quando há tabelas cadastradas
- É necessário estar na aba "Tabelas de Frete" do transportador
- Cada card de tabela tem seu próprio botão de copiar

## 🔍 Não Encontrou?

Verifique:

1. ✓ Você está em **Transportadores** > [Nome do Transportador]?
2. ✓ Está na aba **"Tabelas de Frete"** (não "Visão 360")?
3. ✓ Existem **cards de tabelas** na tela?
4. ✓ Os cards têm **4 ícones** no canto superior direito?
5. ✓ O **2º ícone é azul**?

Se sim para todos, você encontrou! Se não, recarregue a página (F5).
